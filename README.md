# Entrega 2 - Hands-on Estrutura de Controle II

Projeto em Java desenvolvido para a disciplina, contendo um menu com diferentes atividades relacionadas a estruturas de controle.

## Funcionalidades

O programa possui um menu com as seguintes opções:

1. Distância entre pontos  
Calcula a distância entre dois pontos com base em latitude e longitude.

2. Contador de caracteres  
Conta letras, números, espaços e outros caracteres em uma frase digitada pelo usuário.

3. Pergunta de múltipla escolha  
Apresenta uma questão sobre UX com até 3 tentativas de resposta, utilizando `switch` e `do-while`.

4. Print  
Exibe um desenho feito com caracteres no terminal.

0. Sair  
Encerra o programa.

- Java
- VS Code
